1. O que é o Node.js?
é um ambiente de execução (runtime) JavaScript de código aberto e multiplataforma que permite executar códigos JavaScript no lado do servidor (backend), fora do navegador
2.  programaO Node.js usa qual linguagem deção?
Node.js usa a linguagem de programação JavaScript [1, 2]. 
3. O Node.js roda no navegador ou fora dele?
Embora o JavaScript tenha siara que serdo criado originalmente para rodar apenas dentro de navegadores (como Chrome, Firefox, Safari) para tornar páginas web interativas, o Node.js foi criado para permitir que o JavaScript seja executado diretamente no sistema operacional. 
4. Pra que serve o Node.js?
principalmente para executar código JavaScript no lado do servidor (back-end), permitindo o desenvolvimento de aplicações web rápidas, escaláveis e em tempo real.

6. O que é o npm ?
 npm (Node Package Manager) é o gerenciador de pacotes padrão para o ambiente de tempo de execução Node.js e o maior registro de software do mundo
7. O que significa a palavra módulo no Node.js?
é uma unidade funcional de código, geralmente representada por um arquivo JavaScript (ou uma pasta contendo múltiplos arquivos), que encapsula funcionalidades específicas para serem reutilizadas em diferentes partes de uma aplicação. 
8. O Node.js é mais usado no front-end ou no back-end ?
O Node.js é utilizado majoritariamente no back-end. Ele é um ambiente de execução que permite rodar JavaScript no servidor, sendo ideal para APIs de alta performance, aplicações em tempo real e microsserviços.
9. O que é o arquivo package.json ?
O package.json é um arquivo central em projetos JavaScript/Node.js que funciona como um "RG" do projeto, contendo metadados (nome, versão, descrição), gerenciando dependências (pacotes externos), definindo scripts para automação. 
10. O Node.js é gratuito ou pago?
 Node.js é totalmente gratuito e de código aberto (open-source).
